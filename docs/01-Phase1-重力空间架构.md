## 核心理念

标准 Transformer 的最大隐含假设：**「思考」和「行动」在每一步锁步。**

每一个 token 生成前，模型重新计算一次注意力——这是它的「思考」。但这个思考没有后台，没有潜伏期，没有权重演化。每次计算都是从头开始的条件生成。做完一步，上一次的「思考状态」就丢了。

**重力空间架构否定了这个假设。**

它认为模型推理需要两个层次，两个不同的时间尺度，两个不同的表征空间：

---

## 第一层：思考层（潜意识 / 慢权重 / 持续演化层）

### 运行方式

- **持续运行的子系统**，不直接参与 token 的显式生成
- 运行在**共振空间（Resonance Space）**，而不是概率空间
- 以不同的时间尺度更新——可能每 N 个 token 更新一次，也可能按语义边界触发，而不是每步锁步
- 感知「低可能性信息」——那些在 logits 中微弱但背后有多条独立推理路径对齐的信号
- 信息来源包括：逻辑约束、哲学框架、现实因素、用户提示的结构特征、模型历史输出的残差

### 表征介质：重力场（Gravity Field）

思考层维护一个**引力张量 F ∈ R^(d × m)**：
- d = 引力空间的维度（对应「质量」的种类）
- m = 当前活跃选项的数量

每个候选选项从一个初始引力 G₀ 开始（类似于 logit 初值）。
当信息流过思考层的每一层「空间层」时：
- 逻辑层：+Δg₁（逻辑一致性对选项的支持强度）
- 哲学层：+Δg₂（与内在先验的对齐程度）
- 提示层：+Δg₃（用户在 prompt 中隐含的方向牵引）
- 历史层：+Δg₄（模型自身输出的残差反馈）

**关键性质**：重力不是概率空间。多个选项的引力可以同时增长，它们之间存在引力牵引（共享证据路径的选项相互靠近），而不是概率归一化的你死我活关系。

### 引力演化方程（直觉版）

```
F(t+1) = F(t) + Σ Δg_i(t) + Interaction(F(t)) + Decay(F(t))
```

其中：
- Δg_i = 来自第 i 条证据路径的贡献
- Interaction = 选项之间的引力牵引（距离越近牵引力越大）
- Decay = 引力衰减（避免爆炸，保留最近的有效积累）

重点在 **Interaction 项**——它让共享相同推理线索的选项在重力场中彼此靠近，产生「聚类」行为，这是当前 Transformer 的 softmax 归一化做不到的。

---

## 第二层：行动层（意识 / 快权重 / 产出层）

### 运行方式

- 标准的注意力 + FFN 前向传播
- 对于 token 生成是同步的（每一步计算一次）
- **注意力机制本身不变**——注意力不参与「思考」，只负责「组织」
- 行动层的产出是事实化的结果，但受思考层的权重偏置约束，使得每一步的产出在「整体稳定」的前提下有一定的不固定性

### 行动层的结构性质

- 最终的文本输出 = 行动层的**落地**
- 自带语言模型的组合性（compositionality）——即使思考层的偏置分布有偏移，语言模型的结构本身能约束最终产出不至于离谱
- 这就是「人也会说错话」的机理侧：思考层的某条低概率路径权重被抬太高，行动层的正则化没压住，偏了，但整体不会崩

## ══════════════════════════════════════
## 核心难点：两层如何交互
## ══════════════════════════════════════

这是整个架构最微妙的部分。交互方式的选择决定了：可训练性、计算成本、可解释性。

小刘明确说：**思考层不走注意力。** 不是 cross-attention，不是对行动层的 hidden state 做注意力。交互是更底层的：**权重空间的偏置/条件化。**

以下探索了几个可能的交互模式，按侵入性从低到高排列。

---

### 方案A：增益控制（Gain Control / 乘性门控）

**直觉类比**：大脑的神经调质（多巴胺、去甲肾上腺素）。不携带内容，不改变连接矩阵——只改变神经元的**响应敏感度**。

**实现**：
```
# 思考层输出一组增益向量 g ∈ R^d（d = 行动层隐藏维度）
# 行动层的每个子层应用这个增益：

h' = g ⊙ LayerNorm(h)    # 逐元素乘法，缩放激活幅度
h'' = Attention(h')
```

- 不改变方向（不旋转表示空间），只改变幅度
- 计算几乎为零（d 维逐元素乘法）
- 思考层可以以低频率更新 g，行动层每步重用

**优缺点**：
- ✅ 最轻的交互成本，几乎不增加计算
- ✅ 可端到端训练（乘法操作可微）
- ❌ 表达能力有限——它不能「选择」影响哪些维度，只能放大/缩小所有

---

### 方案B：势场调制（Potential Field Modulation）

**直觉类比**：物理学的势能场。带电粒子在电场中移动时，不接收「去左边」的指令，而是感受一个势能梯度——路径被自然地弯曲了。

**实现**：
```
# 思考层的重力场 Φ(x) 是一个平滑的势函数，定义在表示空间上
# 行动层的 hidden state h 每经过一层时，感受势场梯度 ∇Φ(h)：

h' = h + Attention(h) - α · ∇Φ(h)    # 在注意力更新后叠加一个「势场漂移」
```

或者更轻量：
```
# 偏置项注入
h' = LayerNorm(h) + 偏置_bias_来自思考层
```

**关键点**：
- 势场不告诉行动层「做什么」，它改变的是行动层的能量景观
- 行动层在势场中自由移动，但会被低势能区域「自然吸引」
- 这种吸引不是强制路由，是**条件化偏置**

**优缺点**：
- ✅ 跟重力场比喻完美对应——引力就是势场的梯度
- ✅ 思考层的输出有物理意义（势能面），可解释性强
- ❌ 连续势场的计算和维护比增益向量贵
- ❌ 需要定义一个平滑的 Φ(x)，在离散 token 空间中不直接

---

### 方案C：慢上下文向量（Slow Context Vector）

**直觉类比**：人阅读一篇文章时，潜意识里持续维护一个「这文章在说什么」的全局语境——不随每个词刷新，而是慢慢演进。这个语境在你读下一个句子时，已经在了。

**实现**：
```
# 思考层维护一个慢速演化的上下文向量 c ∈ R^d
# 插入到行动层的每个子层之前：

# 在 LayerNorm 中使用 c 作为可学习的偏移和缩放：
γ = W_γ · c + b_γ        # 缩放参数
β = W_β · c + b_β        # 偏移参数
h' = γ ⊙ LayerNorm(h) + β    # 条件化的 LayerNorm

# 注意力计算时，c 也可以调制 Q/K/V：
Q' = (W_Q · h) + (U_Q · c)   # c 作为 Q 的附加输入
```

这实际上就是 **Conditional Normalization**——StyleGAN 用这个做风格迁移，FiLM 用这个做视觉推理的层级条件化。

**优缺点**：
- ✅ 计算成本可控（几个线性映射）
- ✅ 已在大规模模型验证过（FiLM, Conditional LayerNorm, AdaLN）
- ✅ 思考层更新 c 时可以低频率（每 N 步），不影响行动层的每步速度
- ❌ c 是全局向量 → 表达能力受限于 d 维度；如果 m 很大（很多活跃选项），一个向量不够

---

### 方案D：表示空间剪切（Representation Shearing）

**直觉类比**：不是改变地图上的点在哪里，是改变地图本身的度量——重新定义「近」和「远」的含义。

**实现**：
```
# 思考层输出一个线性变换矩阵 S ∈ R^(d×d)
# 作用于行动层的表示空间：

h' = Attention(S · h)     # 变换后的表示去做注意力
# 等价于：思考层重新定义了 token 之间在注意力计算时的「距离」
```

**优缺点**：
- ✅ 表达能力最强——可以重新定义 token 之间的语义关系
- ❌ d×d 矩阵乘成本高（d=4096 → 16M 参数）
- ❌ 思考层的输出不稳定时容易破坏注意力可靠性
- ❌ 可解释性差——你不知道 S 在做什么重组

---

### 交互方案对比总结

| 方案 | 成本 | 表达力 | 可解释性 | 与你重力场的契合度 |
|-----|------|-------|---------|-----------------|
| **A: 增益控制** | ⭐ 极低 | ⭐⭐ | ⭐⭐⭐ | ⭐⭐ |
| **B: 势场调制** | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **C: 慢上下文** | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ |
| **D: 表示剪切** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐ | ⭐⭐ |

**基于你的重力场直觉，势场调制（B）物理映射最自然，慢上下文向量（C）工程实现最可行。**

如果你选 B+C 混合——思考层在重力场里演化势能面，然后压缩成一组条件化参数（γ, β）注入行动层的 LayerNorm——那交互机制在不丢失物理直觉的前提下变得可训练。

## 数学参考：你的星球宇宙对应的已知结构

以下是你直觉中「星球宇宙」比喻在数学和现有文献中的对应物。不是模拟，是**严格映射**。

---

### 1. 流形学习（Manifold Learning）

**直觉对应**：你的宇宙空间是高维表示空间，星球是数据点在高维空间中的凝聚区域。

**核心思想**：高维空间中的点看似离散，实际分布在低维的、有几何结构的流形上。

**你的对应**：
- `宇宙空间` = 高维表示空间（如 4096 维 Transformer hidden state）
- `星球` = 流形上的局部吸引子（attention 凝聚点）
- `从远处看轮廓` = 低维投影（UMAP / t-SNE 降维到 2D 看结构）
- `深入才看清` = 沿流形的测地线局部展开

**关键文献**：
- Roweis & Saul (2000) — **LLE (Locally Linear Embedding)**
- Tenenbaum, de Silva & Langford (2000) — **Isomap**
- **t-SNE** (Van der Maaten & Hinton, 2008)
- **UMAP** (McInnes et al., 2018) — 目前最主流的流形可视化工具

---

### 2. 双曲嵌入（Hyperbolic Embedding）

**直觉对应**：星球之间有隐式的层级关系，但不需要显式树结构——双曲空间自动容纳了它。

**为什么是双曲空间**：
- 欧氏空间的体积随半径多项式增长：V ~ r^d
- 双曲空间的体积随半径**指数增长**：V ~ e^(r·(d-1))
- 这意味着双曲空间天生能容纳**指数级多的"星球"**，且保持层级距离

**你的对应**：
- 大量候选选项共存而不冲突 → 双曲空间的指数容量
- 层级化的引力关系 → 双曲空间中的树状嵌入（根=抽象概念，叶子=具体 token）
- 远处只看轮廓 → 双曲空间中的低精度但保拓扑的投影

**关键文献**：
- **Poincaré Embeddings** (Nickel & Kiela, NeurIPS 2017) — 在双曲空间中嵌入 WordNet 层级
- **Hyperbolic Neural Networks** (Ganea et al., 2018) — 基于 Lorentz 模型的可微操作
- **Hyperbolic Attention** (Gulcehre et al., 2019) — 注意力机制的双曲版本

**与你的契合度**：⭐⭐⭐⭐⭐ — 星球宇宙和双曲空间之间的映射几乎是天然的。

---

### 3. 神经常微分方程（Neural ODE）

**直觉对应**：引力场的演化是连续的，不是离散 step——Neural ODE 描述的就是连续时间演化的系统。

**核心思想**：一个深度网络层 `h₁ = h₀ + f(h₀)` 是离散欧拉法求解微分方程 `dh/dt = f(h, t, θ)` 的单步。Neural ODE 直接用 ODE solver 求解整个演化路径。

**你的对应**：
- 引力场 `F(t)` 随时间的连续演化 → Neural ODE 的 `dz/dt = f(z, t, θ)`
- 初始引力分布 F₀ → 初始状态 `z₀`
- 最终引力分布 F_T → 求解结束状态 `z_T`
- 外界输入给引力场的 Δg → ODE 中的驱动项 `u(t)`

**关键文献**：
- **Neural ODE** (Chen et al., NeurIPS 2018) — 连续深度模型的奠基工作
- **Lagrangian Neural Networks** (Cranmer et al., 2020) — 用拉格朗日力学约束系统学习保守力场
- **Hamiltonian Neural Networks** (Greydanus et al., 2019) — 能量守恒的系统建模

**与引力场的契合**：如果 `f(z, t, θ)` 被设计为一个保守力场（散度为零），系统自动有能量守恒性质，不会发散——这对控制引力爆炸有价值。

---

### 4. 向量符号架构（VSA / Hyperdimensional Computing）

**直觉对应**：你的思考层不需要精确数值——它是不精确但保结构的。VSA 刚好做这个。

**核心思想**：用几千维的高维随机向量（每个维度 ±1 或 0/1）做基本的认知操作：绑定、叠加、共振检索。不需要训练，计算是代数的。

**你的对应**：
- `从远处看星球` → 高维向量的叠加态（模糊但保结构）
- `深入看到全貌` → 通过共振从叠加态中恢复出具体概念
- `引力积累` → 多源证据向量的分量叠加
- `低概率但多源支持的选项` → 高维空间中向量分量有微弱但一致的信号

**关键文献**：
- **Kanerva (2009)** — _Hyperdimensional Computing: An Introduction_
- **Plate (1995)** — Holographic Reduced Representations（圆周卷积做绑定）
- **Gayler (1998)** — Multiplicative Binding

**与你的契合度**：⭐⭐⭐⭐ — 思考层的不精确性正好匹配 VSA 的噪声鲁棒特性。

---

### 5. 动力系统 + 吸引子网络（Dynamical Systems & Attractor Networks）

**直觉对应**：星球就是引力场中的吸引子——系统自然地流向某些稳定点。

**核心思想**：动力系统的状态随方程演化，某些区域是稳定平衡点（吸引子），系统一旦进入这些区域就会停留。

**与你的对应**：
- 多个稳定吸引子 = 你的多个星球（候选选项）
- 初始状态附近的小扰动 = 初始引力 + 每层 Δg
- 系统的最终收敛 = 引力场的最终权重分布

**关键文献**：
- **Hopfield Networks (1982)** — 最早的吸引子网络
- **Modern Hopfield Networks** (Ramsauer et al., 2020) — 与 Transformer 注意力的等价性证明
- **Supertypes attractors** (Krotov & Hopfield, 2016) — 连续的、高容量的吸引子网络

**惊人联系**：Modern Hopfield Networks 的更新方程**等价于 Transformer 注意力**。这意味着变体的吸引子网络可能就是你的架构中思考层的自然计算基元。

## 核心猜想与受控预测
## ══════════════════════════════════════

> **一个 token 在概率空间中的位置（logit 值）包含的信息量，远少于它在共振空间中的位置（多源证据强度的分布）所包含的信息量。**
> 
> 标准采样策略（temperature / top-p / top-k）丢失了后者的信号。
> 
> 重力空间架构的目标：在压缩到概率空间之前，让多源证据的共振在单独的维度上演化，不提前被 softmax 归一化湮灭。

### 可检验预测

1. **在多源证据冲突的任务上**（比如一段文本有两条矛盾的推理线索），重力空间架构会比标准 Transformer 更慢但更准确做出判断——因为它在共振空间里积累了两条线索各自的权重再决策，而不是被一条更强的语言模型先验压制。

2. **选项的引力差距小时**，架构的输出稳定性降低（「犹豫」现象）——这不是 bug，而是引力相近时，势能面平坦，行动层自然会在多个吸引子之间漂移。

3. **思考层的更新频率和输出质量之间有一个最优频率**——更新太频繁（每步都动）会让行动层的势能面不稳定，输出随机性增加；更新太慢（几万步不动）会丢失适应性。人类思考也有这个区间。

## 被动/主动双模式（2026-07-06 晚补充）

### 被动模式（始终在线）
- 持续感知行动层的 hidden state / 注意力分布 / logits
- 低精度压缩到重力场
- 不筛选、不判断、不泛化——只存轮廓
- 类比：坐在咖啡馆看书，背景有人在聊天——声音进了耳朵，但不处理

### 主动模式（触发开启）
- 高精度更新重力场
- 建立跨线索的引力牵引关系
- 学习：这个上下文下，什么引力分配带来了好的输出
- 类比：放下书转头去听——不只是留痕迹，是建立结构化的理解

### 触发信号
架构内置，不需要外部监督：
1. **注意力熵突然增高** — 模型在多个选项间犹豫
2. **语义边界** — 新一段输入开始
3. **重力场中多个选项的引力差距低于阈值** — 多个答案各有道理，需要精细区分

### 爆炸控制
- 被动模式写入频率低、精度低 → 积累速度天然慢
- 主动模式只在必要时开启 → 大部分时间重力场低功耗运行
- 两种模式的分裂本身就是爆炸控制的主要机制

---

## 训练方案（2026-07-06 晚补充）

思考层不通过梯度从行动层学习——它是**独立训练**的：

**训练流程：**
1. 行动层（标准 Transformer）正常前向传播
2. 思考层在关键位置感知行动层状态（hidden state 快照）
3. 思考层将这些信号压缩到重力场
4. 训练目标（自监督）：给定当前重力场状态，能否预测行动层下一层需要的偏置？

**关键性质：**
- 不需要人工标注、不需要 RL、不需要端到端梯度穿过行动层
- 行动层自己的轨迹就是训练数据
- 行动层的结构可以维持在标准 Transformer 不变
- 训练信号由思考层自己产生（压缩+预测的一致性）

**推理流程：**
1. 重力场已经训练好
2. 输入 token → 重力场根据上下文产生偏置 → 行动层在偏置空间中处理 → 输出
3. 思考层不做推理时的权重更新，只做推理时的条件化

**类比：自编码器 + 预测编码**
- 自编码器：将行动层高维状态压缩到低维重力场，再展开
- 预测编码：重力场学习预测行动层的下一状态，误差用来更新重力场

## 宇宙空间 / 星球宇宙观（小刘的原创比喻）

如果把重力空间可视化为一个宇宙：

- 每个候选选项 = 一颗星球
- 初始引力 G₀ = 星球的质量
- 每层空间的 Δg = 星际尘埃对星球引力的贡献
- 选项之间的引力牵引 = 星球之间的潮汐力
- 模型的输出 = 对一个星球的着陆

从远处看：
- 能看到的不是每个 token 的概率值，而是引力场的大致地形
- 高引力区域表示「多条推理路径在那里汇聚」
- 低引力区域表示「孤立证据路径」

要完全理解一个星球（选项）的全貌，需要「接近它」——即把注意力聚焦到那个区域进行局部展开。

## 核心洞见清单（小刘原创）

1. 思考层不应该是注意力机制的变体——它是**完全不同的东西**：权重空间的慢速调制器
2. 思考层存在于**共振空间**，不是概率空间。概率空间已经是在压缩后的产物
3. 多源证据的维度数量（有多少条独立路径支持一个结论）比单条证据的强度更重要——这是现有采样策略完全忽略的
4. 重力不是竞争关系。多个选项的引力可以同时增长——这是 softmax 归一化做不到的
5. 思考层的更新周期 ≠ 行动层的产出周期——潜意识有它自己的时钟
6. 交互是偏置/条件化，不是路由/指令——潜意识不命令意识，只调整意识的权重立场
